# Comment supprimer son compte

## 1ère étape 
   Connectez vous à votre compte

## 2ème étape
   Allez sur le menu et cliquez sur le bouton paramètres.
![Screenshot](/img/menu_parametres.JPG)

## 3ème étape
   Appuyez sur le bouton supprimer le compte.
   Attention si vous êtes un livreur vous devez avoir fini vos livraison en cours.
![Screenshot](/img/supprimer.JPG)
