Comment fonctionne le rôle de producteur

Le rôle du producteur est d'imprimer des qrcode sur les commandes qu'il doit donner au livreur.

1ère étape

Cliquez sur le bouton mes commandes.

2ème étape

La liste de vos commandes en cours sont visible sous forme d'une liste. Si vous avez aucune commandes vous serez redirigé à la page d'accueil.

3ème étape

Cliquez sur le bouton generer un code pour créer le qrcode qui correspond à la commande. NB : nous pouvons voir l'état de la commande comme étant selectionne par le livreurs.

4ème étape

Cliquez sur le bouton imprimer puis une fenêtre va s'afficher. Cliquez ensuite sur imprimer.

5ème étape

Agrapher le qrcode sur la commande à livrer.

6ème étape

Attendez que le livreur arrive pour lui donner la commande à scanner.
