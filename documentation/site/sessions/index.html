  • Les sessions

Pour permettre aux utilisateurs de rester connecter entre les pages, nous utilisons des variables de sessions.

Ces variables de sessions permet de garder en mémoire seulement les infos importantes sur l'utilisateur:

  • session.connected: cette variable est un boolean, il prend la valeur vrai ou faux, elle permet de changer l'interface du menu et les fonctionnalitées.

  • session.name: Cette variable est le nom de l'utilisateur

  • session.surname: Cette variable est le prénom de l'utilisateur

  • session.mail: Cette variable est le mail de l'utilisateur

  • session.phone: Cette variable est le numéro de téléphone de l'utilisateur

  • session.password: Cette variable est le mot de passe chiffré de l'utilisateur

  • session.role: Cette variable est le role de l'utilisateur
